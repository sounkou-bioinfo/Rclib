# Rclib

[clib](https://github.com/clibs/clib), the C package manager CLI.

This package bundles the clib command-line tool, allowing R it's distribution as R package and likely use in R projects that use the header libraries provided in the clib organization.

## Reference

- [clib on GitHub](https://github.com/clibs/clib): Package manager for the C programming language. Lets you fetch and manage stand-alone "micro" C libraries, promoting smaller utilities and easier discovery. see [clib documentation](https://github.com/clibs/clib) for more details.
